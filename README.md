# Intérprete de Lenguaje de Programación
**Universidad del Istmo**   
**Materia:** Teoría de Algoritmos y Complejidad

## Descripción 
Este proyecto implementa el ciclo de vida completo de un lenguaje de programación imperativo. 

## Especificaciones
* **Lenguaje Base:** Python 3.14.3.
* **Librería:** PLY (Python Lex-Yacc).
* Gestión de Código:GitHub.

## Entregables Incluidos
Código fuente modular (`lexer.py`, `parser.py`, `interpreter.py`, `main.py`).
* Scripts de prueba (`variables.txt`, `test_condicionales.txt`, `test_ciclos.txt`) para demostrar el funcionamiento de ciclos y condicionales.
